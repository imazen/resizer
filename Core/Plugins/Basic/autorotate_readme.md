# AutoRotate plugin (v3.1+)

Automatically rotates images based on the EXIF Orientation flag embedded by the camera. 

## Installation

1. Add `<add name="AutoRotate" />` to the `<plugins />` section.

## Syntax

Add `&autorotate=true` to any image url.
